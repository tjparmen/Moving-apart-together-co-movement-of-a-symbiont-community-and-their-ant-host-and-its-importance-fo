DATA & FILE OVERVIEW

The data is stored in three .txt files

colonization.txt

	Locatie: two sites (categorical data)
	ID: nest id (categorical data)
	id2: resorting id (categorical data)
	Iddef: nest id labelling used in analyses (categorical data)
	Treatment: old vs new (categorical data)
	Replica: (numerical)
	other columns contain count data for each myrmecophile


coordinates.txt
	
	nest: nest id (categorical data)
	site: two sites (categorical data)
	Iddef: nest id labelling used in analyses (categorical data)
	sampling: sampling year
	type: budded or original (categorical)
	a: nest length (numeric)
	b: nest width(numeric)
	surface: nest surface (numeric)
	x: x coordinate
	y: y-coordinate
	UTM1: UTM coordinate
	UTM2: UTM coordinate


pitfalls_def.txt
	
	datum: sampling date
	set: set of nests (categorical data)	
	replica: replicate (categorical)
	nestkoppel: adjacent nests (categorical)	
	nest: nest id (categorical)	
	treatment: periphery, trail or extranidal (categorical)		
	distance: distance pitfall to nest (numerical)	
	Redundant: x or N (categorical)	
	other columns contain count data for each myrmecophile

Missing data codes: NaN (data not available)
